To rebuild project:
- Download archive (this is NetBeans project)
- Execute Database.sql (dump of database)
- Lunch programm (in will generate .ser file in project file folder - it handles defauld settings for database connection)
- At "Login" frame press "Connection..." and check if you are connected to database. (if not - change settings)
